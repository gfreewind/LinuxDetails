# 4.reuse_addr_option


## 问题


什么时候需要使用SO_REUSEADDR?


## Answer


当proto, addr, 和port三个元素都相同时，需要使用SO_REUSEADDR。

其中INADDR_ANY表示任意地址，其与所有地址视为相同。
